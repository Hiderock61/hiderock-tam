# PROJECT_CARD

## このアプリは何？
- ヒデロックテンプレ法©️（Hiderock Template Method）を実践する軽量ウェブアプリです。
- ユーザーが「素材」を入力し、以下の工程に沿って記述を整理することで、最終的に「AIに渡す文（方法©️文）」を生成・コピーできるツールです。
  - 拾う → 名前をつける → レイヤー分解 → 地上語へ翻訳 → 方法©️化
- 実装はクライアント側の静的ページで完結しています（index.html / script.js / style.css）。

## 現在の状態

## 主な機能
- 素材入力台帳（自由テキスト）
- 5ステップ工程カードの入力欄（拾う / 名前 / レイヤー分解 / 地上語へ翻訳 / 方法©️化）
- 劇団員検査レイヤーのチェック欄（複数の視点での確認）
- 生成ロジック（script.js）による「方法©️文」組み立て
- 生成文のクリップボードコピー機能
