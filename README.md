---

## 概要

ヒデロックテンプレ法©️ は、人生の素材を、AIと現実で使える工程に変換するための方法©️です。

対象そのものではなく、  
**対象を扱うための工程** を保存します。

そのため、ジャンルが変わっても同じ工程を再利用できます。

---

## 対応ジャンル

この方法©️は、以下のような対象に使えます。

- アプリ開発
- note記事
- 病院説明
- クラウドワークス案件
- ロードー判断
- ギター練習
- 食べ物ログ
- 呼吸体験
- 入院バケーション
- スパイスと世界史
- AIへの依頼文
- GitHub Pages用プロトタイプ
- 収入導線の整理

---

## 基本工程

1. **拾う**  
   変な言葉・違和感・体感・出来事を拾う。

2. **名前をつける**  
   呼べる形にする。

3. **レイヤー分解する**  
   身体、現実、創作、仕事、AI、社会語に分ける。

4. **地上語へ翻訳する**  
   他人やAIに渡せる言葉にする。

5. **方法©️化する**  
   次回も使える工程として保存する。

| 🦅 競争手を見る鷹 | 市場の混雑と差別化を見る |
